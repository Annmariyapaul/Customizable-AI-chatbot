# Customizable AI Chatbot

A simple AI chatbot built with Gradio in Google Colab. The chatbot allows users to customize the system prompt and adjust the temperature to control the chatbot's personality, response style, and creativity.

## Features

- Customizable System Prompt
- Adjustable Temperature Slider
- Real-time Response Streaming
- Maintains Chat History
- User-friendly Gradio Interface

## Technologies Used

- Python
- Gradio
- OpenAI API
